VCP 波動收縮

  • 趨勢站上 SMA50 / SMA200
  • 2 至 4 次收縮且幅度遞減
  • Pivot 突破搭配 1.5 倍量
  • 跌破 Pivot / EMA10 / EMA20 / SMA50 出場

35 法則

  • 3/5 EMA 金叉與 35 日突破
  • 3% 至 5% 回檔靠近 EMA 支撐
  • 乖離過高、RSI 超買、3/5 EMA 死叉出場
  • 預設 35% 分批停利

短線 / 當沖

  • EMA5 / EMA10 / EMA20 順勢排列
  • MACD 金叉且 RSI 未過熱
  • 開盤突破、VWAP 回踩、區間快進快出
  • 每筆風險 0.5% 至 1%

長線 / 趨勢

  • 價格站上 SMA200
  • SMA50 / SMA100 / SMA200 多頭排列
  • 長週期 MACD 金叉
  • 跌破 SMA50 或 MACD 死叉降風險
